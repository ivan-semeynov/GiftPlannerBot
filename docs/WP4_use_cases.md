# Use Cases

## UC-1: Подбор подарка
1. Пользователь инициирует подбор
2. Вводит параметры
3. Получает рекомендации
4. Оценивает варианты

## UC-2: Персонализация
1. Пользователь ставит лайки
2. Система обновляет профиль
3. Рекомендации адаптируются

## UC-3: Повторный запрос
1. Пользователь запускает подбор повторно
2. Учитывается история
