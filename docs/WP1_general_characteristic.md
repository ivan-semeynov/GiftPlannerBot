# General Characteristics of the Project

## Назначение
Проект предназначен для автоматизации и интеллектуализации процесса выбора подарков с использованием методов машинного обучения.

## Тип проекта
- Учебно-прикладной
- ML-first
- Рекомендательная система

## Целевая аудитория
- Пользователи Telegram
- Люди, подбирающие подарки к праздникам
- Пользователи без технической подготовки

## Формат реализации
- Интерфейс: Telegram-бот
- Взаимодействие: диалоговое (FSM)
- Рекомендации: персонализированные

## Ограничения проекта
- Используются только открытые датасеты
- Данные пользователей обезличены
- Модель обучается офлайн
