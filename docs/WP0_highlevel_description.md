# High-Level Description

Данный проект представляет собой ML-ориентированный Telegram-бот для интеллектуального подбора подарков.  
Система использует методы рекомендательных систем и машинного обучения для формирования персонализированных рекомендаций на основе параметров запроса пользователя и истории его взаимодействий.

Проект ориентирован на массового пользователя и решает прикладную проблему выбора подарков в условиях ограниченного времени и неопределённости предпочтений получателя.  
Особенностью проекта является отсутствие rule-based MVP — рекомендательная логика строится сразу на базе ML-подхода.

Проект реализуется в виде Telegram-бота с диалоговым интерфейсом и поэтапной архитектурой, включающей:
- ML-модуль рекомендаций,
- модуль хранения данных,
- диалоговый FSM,
- систему оценки качества рекомендаций.
