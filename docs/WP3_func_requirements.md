# Functional Requirements

## Основные требования
- Пошаговый сбор параметров пользователя
- Формирование рекомендаций с помощью ML-модели
- Персонализация рекомендаций
- Обработка пользовательского feedback
- Объяснимость рекомендаций

## Дополнительные требования
- Быстрый режим подбора
- Сохранение истории взаимодействий
- Настройки пользователя

## Нефункциональные требования
- Быстрый отклик
- Масштабируемость
- Модульность архитектуры
