# Problem Statement

## Описание проблемы
Процесс выбора подарка характеризуется высокой неопределённостью, когнитивной нагрузкой и ограниченностью времени, особенно в предновогодний период.

Пользователь, как правило, не обладает:
- полной информацией о предпочтениях получателя,
- знаниями о релевантных вариантах,
- возможностью быстро сузить выбор.

## Цель проекта
Разработать систему, автоматически формирующую релевантные рекомендации подарков на основе параметров пользователя и его поведения.

## Входные данные
- параметры получателя подарка
- бюджет
- интересы
- ограничения
- история взаимодействий

## Выходные данные
- ранжированный список рекомендаций
- объяснение рекомендаций
